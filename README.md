# SRA2 : XP & Cash Economy

A Foundry VTT module specifically designed for the **Shadowrun Anarchy 2 (SRA2)** system. This module overhauls the default economy by separating Character Advancement (XP) from Monetary Wealth (Cash).

![Foundry v11+](https://img.shields.io/badge/Foundry-v11+-informational)
![Foundry v13](https://img.shields.io/badge/Foundry-v13-success)

## üåü Features

*   **Distinct Currencies**: Replaces the generic "Nuyen / ¬•" occurrences on the character sheet with "XP" for build points, while injecting a dedicated "Cash" field.
*   **Dual Spendable Pools**: In the bottom right corner of the character sheet, you will now clearly see two editable tracking fields side-by-side: your **Spendable XP** and your available **Cash**. The gross build cost of your character is cleverly hidden but visible as a tooltip when hovering over the XP label.
*   **True Zero-Cost System**: Items purchased with Cash are natively forced to `free-equipment` in the SRA2 database upon creation. This keeps your underlying character total XP mathematics perfectly clean without visual hacks!
*   **Automated Purchases**: Dragging and dropping a Cash-enabled item onto a character sheet prompts an confirmation dialog. If accepted, the item's cost is automatically deducted from the character's Cash reserves.
*   **Granular Settings**: Customize exactly which item types use Cash instead of XP via the module settings (Equipment, Weapons, Cyberware, Vehicles, Armor, Cyberdecks). 
*   **Fully Localized**: English and French support built-in.

## üì• Installation

1. Copy the module folder `sra2-xp-cash` into your Foundry `Data/modules/` directory.
2. Launch Foundry VTT, go to your World, and enable the module **SRA2: XP & Cash Economy**.
3. Configure your desired item types in the Module Settings.
